Selectors¶

The selectors are intended to be used in combination with tuners in order to find out and decide which model seems to get the best results once it is properly fine tuned.

In order to use the selector we will create a Tuner instance for each model that we want to try out, as well as the Selector instance.

In [1]: from sklearn.datasets import load_iris

In [2]: from sklearn.ensemble import RandomForestClassifier

In [3]: from sklearn.metrics import make_scorer, r2_score

In [4]: from sklearn.model_selection import cross_val_score, train_test_split

In [5]: from sklearn.svm import SVC

In [6]: from btb.selection import UCB1

In [7]: from btb.tuning.hyperparams import FloatHyperParam, IntHyperParam

In [8]: from btb.tuning.tunable import Tunable

In [9]: from btb.tuning.tuners import GPTuner

In [10]: dataset = load_iris()

In [11]: X_train, X_test, y_train, y_test = train_test_split(
   ....:     dataset.data, dataset.target, test_size=0.3, random_state=0)
   ....: 

In [12]: models = {
   ....:     'RF': RandomForestClassifier,
   ....:     'SVC': SVC
   ....: }
   ....: 

In [13]: selector = UCB1(['RF', 'SVC'])

In [14]: rf_hyperparams = {
   ....:     'n_estimators': IntHyperParam(min=10, max=500),
   ....:     'max_depth': IntHyperParam(min=3, max=20)
   ....: }
   ....: 

In [15]: rf_tunable = Tunable(rf_hyperparams)

In [16]: svc_hyperparams = {
   ....:     'C': FloatHyperParam(min=0.01, max=10.0),
   ....:     'gamma': FloatHyperParam(0.000000001, 0.0000001)
   ....: }
   ....: 

In [17]: svc_tunable = Tunable(svc_hyperparams)

In [18]: tuners = {
   ....:     'RF': GPTuner(rf_tunable),
   ....:     'SVC': GPTuner(svc_tunable)
   ....: }
   ....:

Then we perform the following steps in a loop.

  1. Pass all the obtained scores to the selector and let it decide which model to test.

    In [19]: next_choice = selector.select({
   ....:     'RF': tuners['RF'].scores,
   ....:     'SVC': tuners['SVC'].scores
   ....: })
   ....: 

In [20]: next_choice
Out[20]: 'RF'

  2. Obtain a new set of parameters from the indicated tuner and create a model instance.

    In [21]: parameters = tuners[next_choice].propose()

In [22]: model = models[next_choice](**parameters)

  3. Evaluate the score of the new model instance and pass it back to the tuner

    In [23]: model.fit(X_train, y_train)
Out[23]: 
RandomForestClassifier(bootstrap=True, class_weight=None, criterion='gini',
                       max_depth=3, max_features='auto', max_leaf_nodes=None,
                       min_impurity_decrease=0.0, min_impurity_split=None,
                       min_samples_leaf=1, min_samples_split=2,
                       min_weight_fraction_leaf=0.0, n_estimators=48,
                       n_jobs=None, oob_score=False, random_state=None,
                       verbose=0, warm_start=False)

In [24]: score = model.score(X_test, y_test)

In [25]: tuners[next_choice].record(parameters, score)
