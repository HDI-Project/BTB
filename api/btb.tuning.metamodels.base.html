btb.tuning.metamodels.base module¶

class btb.tuning.metamodels.base.BaseMetaModel[source]¶

Bases: object

BaseMetaModel class.

BaseMetaModel class is an abstract representation of a MetaModel that after being fitted can predict the score that is being expected to be obtained with the hyperparameter configuration proposed.

_MODEL_CLASS¶

Class to be instantiated and to be fited and used to generate predictions. This attribute must be overriden by the child class in order to create an actual model.

Type

type

_MODEL_KWARGS¶

Dictionary with the default kwargs to be given for the self._MODEL_CLASS.

Type

dict

_model_kwargs¶

A dictionary that is used during instantiation of the BaseMetaModelTuner in order to be able to give or set arguments for the model.

Type

dict

_model¶

Instance of self._MODEL_CLASS, defaults to None.

Type
